Selected Projects

FORGE: Force-Aware Multi-Granularity Control for Efficient Humanoid Loco-Manipulation

Under Review

A unified reinforcement learning framework for humanoid loco-manipulation with force-aware control, enabling stable locomotion on uneven terrain, end-effector compliance, diverse loco-manipulation tasks via real-time teleoperation, and autonomous policy deployment.

Project page arXiv Code

GAF: Gaussian Action Field as a 4D Representation for Dynamic World Modeling in Robotic Manipulation

ICRA 2026

Proposed a V-4D-A framework with Gaussian Action Field for direct action inference, showing bi-manual dexterous manipulation capability in RLBench tasks and real-world deployment. Achieved +11.54 dB PSNR improvement and boosted robotic task success rate by 10.33% over SOTA methods.

arXiv Project page Code

SynAgent: Generalizable Cooperative Humanoid Manipulation via Solo-to-Cooperative Agent Synergy

Preprint

A framework for generalizable cooperative humanoid manipulation, enabling synergy from solo-agent training to cooperative multi-agent deployment for complex manipulation tasks.

arXiv Project page Code

Motion Tracking Methods for Dexterous-Hand Manipulation

Master's Research Project

Optimized the retargeting pipeline from MoCap data to dexterous-hand interactions, designed RL reward functions for articulated object manipulation, addressed task expansion via policy distillation, and adapted motion generation to various dexterous hands.
